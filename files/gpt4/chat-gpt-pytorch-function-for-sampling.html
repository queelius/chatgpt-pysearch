Tell me about torch.multinomial

torch.multinomial is a function in the PyTorch library, a popular deep learning framework in Python. The function is used for sampling from a probability distribution specified by the input tensor. It is particularly useful when you need to randomly select elements from a given list or tensor according to their corresponding probability distribution.

Here's the function signature:

python
torch.multinomial(input, num_samples, replacement=False, *, generator=None, out=None)

Parameters:

  • input (Tensor): The input tensor containing probabilities. The tensor can have any number of dimensions, but the last dimension should represent the probability distribution over which you want to sample.
  • num_samples (int): The number of samples to draw from the distribution.
  • replacement (bool, optional): If set to True, the function will sample with replacement, meaning that the same index can be selected multiple times. If set to False (default), the function will sample without replacement, and each index can be selected only once.
  • generator (torch.Generator, optional): A custom random number generator for generating samples.
  • out (Tensor, optional): An optional output tensor to store the result.

Returns:

  • output (Tensor): A tensor of shape (input.size(0), num_samples) containing the indices of the sampled elements.

Example usage:

python
import torch

# A tensor representing the probability distribution
probabilities = torch.tensor([0.1, 0.2, 0.3, 0.4])

# Drawing 2 samples without replacement
samples = torch.multinomial(probabilities, 2, replacement=False)

print(samples)

In this example, samples will be a 1D tensor containing 2 indices randomly drawn from the given probability distribution in the probabilities tensor.
