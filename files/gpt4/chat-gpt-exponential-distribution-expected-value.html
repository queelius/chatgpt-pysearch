Give me python code to compute sample variance.

You can use the following Python code to compute the sample variance of a list of numbers:

python
def sample_variance(numbers):
    n = len(numbers)
    mean = sum(numbers) / n
    squared_diffs = [(x - mean) ** 2 for x in numbers]
    variance = sum(squared_diffs) / (n - 1)
    return variance

# Example usage:
sample_data = [0.21897624, 1.93018817, 0.74535197, 0.16373022, 0.58860134, 1.70442671, 0.01260567, 0.19434881, 1.05691792, 2.93102845]

# Function to generate a single λ_hat
generate_lambda_hat <- function(lambda, n) {
  sample <- rexp(n, rate = lambda)
  sample_mean <- mean(sample)
  lambda_hat <- 1 / sample_mean
  return(lambda_hat)
}

# Generate λ_hat values for multiple samples
lambda_hats <- replicate(num_samples, generate_lambda_hat(lambda, n))

# Plot the histogram of the λ_hat values
