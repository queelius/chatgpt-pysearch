You're right that having a suitable computer is important for working with GPT language models. Here are some things to consider:

  1. Compute requirements: GPT models can be computationally intensive, particularly when fine-tuning them on new datasets. You'll need a machine with a powerful GPU (Graphics Processing Unit) to train GPT models efficiently. Ideally, you would want a machine with a high-end NVIDIA GPU, such as a GTX 1080 or RTX 2080 Ti.

  2. Storage requirements: GPT models are also large and can require a lot of storage space. Make sure you have enough hard drive space to store the model files and any training data.

  3. Cost: Building a computer with a high-end GPU can be expensive. However, it may be a worthwhile investment if you plan to do a lot of work with GPT language models. Alternatively, you can use cloud computing services such as Amazon Web Services (AWS), Google Cloud Platform (GCP), or Microsoft Azure. These services offer GPU instances that you can rent on an hourly or monthly basis. This can be a cost-effective option if you don't want to invest in a dedicated machine.

  4. Iteration time: If you decide to use a cloud computing service, make sure you choose an instance type that is suitable for your needs. You don't want to end up waiting hours for your model to train or generate results. Additionally, you'll want to make sure that the service you choose allows you to save and reload your model checkpoints easily, so you can resume your work quickly.

Ultimately, the decision to use a dedicated machine or cloud computing service will depend on your budget and specific requirements. If you have the budget and plan to do a lot of work with GPT, building a dedicated machine may be the best option. Otherwise, cloud computing services can be a cost-effective way to get started.

If you're only using pre-trained language models and not fine-tuning them on new data, the compute requirements will be significantly lower compared to fine-tuning. Inference, which is the process of using a pre-trained model to generate text or make predictions, is generally less computationally intensive than training.

For example, the largest pre-trained GPT-3 model has 175 billion parameters, but it can run on a single NVIDIA A100 GPU. If you're not fine-tuning the model and just using it for inference, you could get by with a lower-end GPU, such as an NVIDIA GTX 1080 or an NVIDIA T4.
