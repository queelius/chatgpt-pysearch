How can I visualize the bias of an MLE estimator?

To visualize the bias of a Maximum Likelihood Estimator (MLE), you can use a simulation approach. Here are the steps:

  1. Choose a probability distribution with a known parameter value. For example, let's say you choose the normal distribution with a true mean of 5 and a true standard deviation of 2.

  2. Generate a large number (e.g., 1000) of random samples from this distribution using the true parameter values.

  3. Estimate the parameter using the MLE method for each of the samples.

  4. Plot a histogram of the estimated parameter values.

  5. Calculate the mean of the estimated parameter values and compare it to the true parameter value. The difference between the mean of the estimated values and the true parameter value is the bias.

  6. Repeat steps 2-5 multiple times to see how the bias changes with different sample sizes and different true parameter values.

By repeating these steps, you can visually see the distribution of estimated parameters and how they differ from the true parameter value. This can help you understand the bias of the MLE estimator.

Sure, here's an example R code that demonstrates the visualization of the bias of an MLE estimator for the mean parameter of a normal distribution:

r
# Set the true parameter values
mu_true <- 5
sigma_true <- 2

# Set the sample size and number of simulations
n <- 100
n_sim <- 1000

# Create an empty vector to store the estimated parameter values
mu_est <- numeric(n_sim)

# Simulate n_sim samples from the normal distribution with the true parameter values
for (i in 1:n_sim) {
  x <- rnorm(n, mean = mu_true, sd = sigma_true)
  
  # Estimate the mean parameter using the MLE method
  mu_est[i] <- mean(x)
}

# Plot a histogram of the estimated mean parameter values
hist(mu_est, breaks = 20, freq = FALSE, main = "Histogram of Estimated Mean Parameter")

# Calculate the bias of the MLE estimator
bias <- mean(mu_est) - mu_true
cat("Bias of the MLE estimator:", bias)

In this example, we simulate 1000 samples of size 100 from a normal distribution with a mean of 5 and a standard deviation of 2. For each sample, we estimate the mean parameter using the MLE method, and store the estimated values in the mu_est vector. We then plot a histogram of the estimated mean parameter values and calculate the bias of the MLE estimator as the difference between the mean of mu_est and the true mean parameter mu_true. Finally, we print the value of the bias to the console using the cat function.
